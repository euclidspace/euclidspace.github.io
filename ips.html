Euclid Cordinate

Indoor Positioning System.

Overview :

Euclid's CORDINATE lets users search, navigate and position themselves inside a building. Today’s smartphones are equipped with in-built sensors for determining user gestures and improve overall user-experience of a smartphone. Our system makes use of these sensor units and low-band bluetooth beacons installed near entrances of the building to accurately position the user and help them navigate anywhere within the building where GPS is not an option.

Our system uses AI based model to learn from the phone’s IMU sensor fusion data to determine the user’s acceleration, direction and the floor on which he/she is in, ultimately obtaining localized position of the user inside the building.

Anyone inside a given building can use CORDINATE by installing the app on their smartphone. And anyone having admin access to the web panel of our system can monitor traffic using the app around the building in real-time, as well as perform analytics on visitor behaviour data.
